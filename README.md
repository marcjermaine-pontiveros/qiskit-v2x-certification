# Qiskit V2X Certification

This repository contains materials and resources for Qiskit V2X certification.

## GitHub Pages

This repository is set up to be hosted as a GitHub Page. To enable it:

1. Go to the repository settings.
2. Scroll down to "Pages" in the left sidebar.
3. Under "Source", select "Deploy from a branch".
4. Choose the "main" branch and "/ (root)" folder.
5. Click "Save".

The site will be available at `https://marcjermaine-pontiveros.github.io/qiskit-v2x-certification/`.
